# TicketEscape

`escape.id` のチケット購入ページで、指定時刻に自動でフォーム操作を行う Chrome 拡張です。  
操作は API ではなく、ページ上のボタンやチェックボックスに対する `click()` で実行します。

## 1. まず最初に（重要）

- この拡張は Chrome Web Store からはインストールできません。
- GitHub からダウンロードして、手動で Chrome に読み込みます。
- 使う前に `escape.id` へログインしておいてください。

## 2. できること

- 指定 URL（`https://escape.id/*`）を事前登録
- 指定時刻（JST）に自動実行
- フォーム表示待機後、券種ごとの数量調整
- 同意チェックボックスの自動チェック（必要時）
- 「カートに入れる」ボタンの自動押下
- 失敗時リトライ、実行ログ保存

## 3. 動作環境

- Google Chrome（最新版推奨）
- macOS / Windows
- ローカルに展開したこのプロジェクトフォルダ

## 4. GitHub からダウンロードする方法（利用者向け）

### 4.1 ブラウザでダウンロード（おすすめ）

1. GitHub のこのリポジトリページを開く
2. 緑の `Code` をクリック
3. `Download ZIP` を選ぶ
4. ダウンロードした ZIP を解凍する
5. 解凍したフォルダを開き、`manifest.json` があることを確認する

注意:
- Chrome に読み込むときは、`manifest.json` があるフォルダを選んでください。
- 1つ上のフォルダを選ぶと読み込みに失敗します。

### 4.2 Git コマンドでダウンロード（任意）

```bash
git clone https://github.com/<YOUR_ACCOUNT>/<YOUR_REPO>.git
cd <YOUR_REPO>
```

## 5. Chrome にインストールする方法（手動）

1. Chrome で `chrome://extensions/` を開く
2. 右上の `デベロッパー モード` を ON
3. `パッケージ化されていない拡張機能を読み込む` をクリック
4. 解凍したフォルダのうち、`manifest.json` があるフォルダを選択
5. 拡張一覧に `TicketEscape` が表示されたらインストール完了

## 6. 初期設定（最初に1回）

1. `chrome://extensions/` を開く
2. `TicketEscape` の `詳細` をクリック
3. `拡張機能のオプション` を開く
4. 以下を入力
- `対象URL`: 例 `https://escape.id/xeoxy-org/e-pd/?d=2026-03-10&u=...`
- `実行時刻 (JST)`: チケット発売時刻
- `ウォームアップ秒数`: 例 `120`
- `リトライ上限`: 例 `3`
- `リトライ間隔`: 例 `200,400,800`
5. `フォーム解析` を押す（券種を自動取得）
6. 券種ごとの数量を設定
- 例: `グループチケット = 1`
7. `保存` を押す

保存するとスケジュールが登録されます。

## 7. 実行方法

### 7.1 予約実行

- 保存した実行時刻になると自動実行されます。
- 事前に対象サイトへログインしておいてください。
- 実行直前は PC のスリープを避けてください。

### 7.2 手動実行

1. Chrome ツールバーの拡張アイコンから `TicketEscape` を開く
2. `今すぐ実行` を押す

## 8. 実行結果の確認

- ポップアップで現在状態を確認可能
- 前回実行の成功/失敗を表示
- 失敗時はエラーコード（例: `E_TICKET_NOT_FOUND`）を確認可能

## 9. アップデート方法（新しい版を使う）

### 9.1 ZIP で入れた場合

1. 新しい ZIP を再ダウンロードして解凍
2. `chrome://extensions/` を開く
3. `TicketEscape` の `更新` ボタンを押す
4. うまく反映されない場合は一度 `削除` して、再度「パッケージ化されていない拡張機能を読み込む」

### 9.2 git clone で入れた場合

```bash
cd <YOUR_REPO>
git pull
```

その後、`chrome://extensions/` で `TicketEscape` の `更新` を押します。

## 10. アンインストール方法

1. `chrome://extensions/` を開く
2. `TicketEscape` の `削除` を押す

## 11. よくある問題

- フォーム解析に失敗する:
  - URL が `https://escape.id/*` か確認
  - ログイン状態か確認
  - ページ表示に時間がかかる場合は再実行
- インストール時にエラーが出る:
  - 選択したフォルダ直下に `manifest.json` があるか確認
  - ZIP のままではなく、必ず解凍済みフォルダを指定
- 実行されない:
  - 実行時刻が過去になっていないか確認
  - `保存` 後に時刻を変更した場合は再保存
  - PC がスリープしていないか確認
- 数量が合わない:
  - 券種名の表記ゆれ（空白/記号差）を確認
  - `フォーム解析` 後に再設定

## 12. 注意事項

- 本拡張は対象サイトの DOM 構造変更で動かなくなる場合があります。
- 利用規約に反しない範囲で使用してください。
- この拡張は Chrome Web Store 版ではなく、ローカル読み込み前提です。
